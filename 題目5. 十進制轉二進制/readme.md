題目5. 十進制轉二進制
---
```c
#include<iostream>  
using namespace std;  
int main(){  
    int n=0;  
    int bin[8]={0};  
    cin>>n;  
    if (n<0){  
        n+=256; //將負數移至整數計算
    }

    //由前往後計算並儲存除2的餘數，即為轉換為二進位
    for (int i = 7; i >= 0; i--){  
        bin[i]=n%2;  
        n/=2;  
    } 
    //由後往前輸出餘數，即輸出二進位表示 
    for (int i = 0; i < 8; i++){  
        cout<<bin[i];  
    }  
    cout<<endl;  
}
```
本題範圍介於-128～127之間，十進位-128開始為二進位表示的127往後接續，

所以 為負數時 要用 n+256 來計算。

再利用陣列 bin[8] 由後往前計算並儲存2的餘數,

計算完後並輸出的數字便是二進位表示。
